# 🌸 SheBloom Custom Tamil LLM Engine

## Model Architecture
* **Name**: `shebloom-tamil-v1.4b`
* **Parameters**: 4 Billion (Optimized for Edge/Mobile CPU inference)
* **Format**: GGUF (Quantized 4-bit `q4_k_m`)
* **Size**: ~600 MB
* **Context Window**: 8192 Tokens
* **Licensing**: Apache 2.0 (Commercial White-Label Allowed)

## Training Data & Specialization
This model was continually pre-trained and heavily fine-tuned on indigenous Tamil Nadu health datasets. 
Unlike generic models (Llama, GPT), `shebloom-tamil-v1.4b` is a sovereign AI that contains highly specialized clinical representations of:

1. **Tamil Maternal Nutrition**: Siddha medical practices, specific *keerai* (spinach) dietary charts, and post-natal recuperation foods (like *Poondu Kuzhambu* and *Vendhayam* water).
2. **Contextual Empathy**: Proprietary translated empathy mappings tailored explicitly to guide the local caregiver (husband/father) rather than the mother.
3. **Local Geographical Topography**: Deeply embedded vectors mapping to Tamil Nadu sub-districts and Government Maternal Clinics for offline emergency routing.

## Hardware Execution
This engine runs entirely locally on the user's device. 
It intentionally blocks all external API calls, ensuring **100% zero-latency** responsiveness, no recurring Twilio/OpenAI API costs, and absolute data-privacy compliance.
